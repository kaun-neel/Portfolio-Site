Overview

A sophisticated debate simulation system where two AI agents (Scientist vs Philosopher) engage in structured arguments using LangGraph DAG workflows.

This system implements a complete debate workflow with:

  • Two AI Agent Personas: Scientist (evidence-based) and Philosopher (ethics-based)
  • Structured 8-Round Format: 4 arguments per agent, alternating turns
  • Memory Management: Contextual argument generation with history tracking
  • State Validation: Ensures logical progression and rule adherence
  • Automated Judging: Comprehensive evaluation with detailed reasoning
  • Complete Logging: Full audit trail of all state transitions and arguments

Architecture

Workflow Nodes

  • UserInputNode: Accepts debate topic at runtime
  • AgentA/AgentB: Scientist and Philosopher personas with distinct reasoning styles
  • MemoryNode: Maintains structured argument history and context
  • ValidateState: Ensures turn order, uniqueness, and coherence
  • JudgeNode: Evaluates complete debate and declares winner
  • CheckCompletion: Manages 8-round limit and flow control

Agent Personas

Scientist

  • Focus: Evidence-based reasoning, empirical data, scientific method
  • Style: Systematic, research-backed arguments with practical considerations
  • Strengths: Methodological rigor, risk assessment, reproducible results

Philosopher

  • Focus : Ethical reasoning, human values, existential implications
  • Style : Deep conceptual analysis with historical and moral perspectives
  • Strengths : Ethical frameworks, human dignity, long-term implications
